我的个人网站搭建成功啦

2019-07-07 22:46 By Zexun Luo.

经过很长一段时间的努力，我终于搭建了这个网站。搭建个人网站的想法，在我脑海中已经酝酿许久了，一直没有付诸行动是因为我并没有搭建网站的经验， 同时也是因为还没下定决心。直到前段时间，我在网页设计制作课程大作业里了解到搭建静态网站的相关知识，便下定决心搭建自己的个人网站。
虽然我已经做好了心理准备，但是搭建网站路上遇到的挫折还是超过了我的设想。原以为，搭建静态网站应该比较容易，因为我已经有了一些HTML、 CSS等相关知识。我原本计划购买国内的服务器，但发现国内服务器基本都要实名备案。实名备案虽然耗时也算了，但我因为一些原因暂时不能实名备案。 接着我考虑购买国外的服务器，但发现域名解析也需要备案。就这样，我的网站搭建计划一度搁置。幸运的是，事情转折发生了——我找到了Github Pages。
Github Pages 是面向用户、组织和项目开放的公共静态页面搭建托管服 务，站点可以被免费托管在 Github 上，你可以选择使用 Github Pages 默认 提供的域名　github.io 或者自定义域名来发布站点。也就是说，它解决了我之前服务器、域名的烦恼。虽然这并不是绝对的，它也存在一些局限性， 但对我现在来说确是足够了。接下来我注册好Github,把网站文件都上传到Github，用Github Pages建立了一个网站。
理论上本文到此就可以结束了，但事实并非如此，我的折腾能力超乎想象。我并没有满足这样的网站，因为单纯HTML、CSS文件写网页太麻烦了，也不方便扩展。 于是我了解到了静态网站生成器，比如Hugo、Hexo。本网站就是用Hugo作为后台管理系统的。Hugo是由Go语言实现的静态网站生成器。事实上我是先尝试了入门 更简单、操作更为便捷的Hexo，但是还没开始使用，但是它速度和性能劝退了我。但是后来我发现我还是严重低估了Hugo对我的难度，在初时我便尝试修改主题样式、布局，留下深刻教训。
一番折腾，已是几天几夜后，终于我还是搭建出一个初始版本的个人网站，相对于个人博客，我更喜欢称之为个人空间。在这里，我可以自由地记录生活、学习，分享感悟。虽然并不切合本文的主题，但我仍想高呼一句，海阔凭鱼跃，天高任鸟飞！
